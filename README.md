## 🚀 Instalação

A **Gbitcode Platform** está disponível oficialmente no Registro NPM. Para instalar globalmente em sua máquina, execute:


```bash
npm install -g gbitcode-platform
```



[![NPM Package](https://img.shields.io/badge/NPM-gbitcode--platform-CB3837?style=for-the-badge&logo=npm)](https://www.npmjs.com/package/gbitcode-platform)



## 🛠️ Comandos para usar a plataforma gbitcode

<img width="21" height="20" alt="image" src="https://github.com/user-attachments/assets/dd7c7b2e-55ad-4397-9472-71ca66d0a33c" />

### 1. `gbitcode login <email>`
Estabelece sua identidade global. O CLI lembrará de você em todos os projetos futuros.

```bash
gbitcode login seu-email@gmail.com
```

## 2. gbitcode init
Inicializa o ecossistema Gbitcode na sua pasta atual, criando os arquivos de configuração e ignorando o lixo (node_modules, etc).

```Bash
gbitcode init
```

  <p align="center">
<img width="377" height="110" alt="image" src="https://github.com/user-attachments/assets/6eda3b2b-e3a2-4d0a-a45b-7ee772268791" />

## 3. gbitcode commit "<mensagem>"
A "mágica" acontece aqui. Transmite todos os seus arquivos para o mainframe com uma barra de progresso e autoridade de e-mail.

```Bash
gbitcode commit "primeiro envio do projeto"
```

## 4. gbitcode clone <nome-do-projeto>
Baixa qualquer projeto do seu mainframe diretamente para a sua máquina atual, reconstruindo todas as pastas.

```Bash
gbitcode clone meu-projeto-web3
```

## 5. gbitcode help
Quer Conhecer Os Comandos? O manual de instruções integrado te ajuda a navegar por todas as funcionalidades da plataforma em segundos.

```bash
gbitcode help
```

# 🧬 Gbitcode Platform

> **Onde seu código ganha vida.** Uma plataforma de versionamento e armazenamento inspirada no GitHub, focada na simplicidade do CLI e transparência pública.

---

## 🚀 Como funciona?
O Gbitcode permite que você envie seus projetos inteiros para a nuvem através de comandos rápidos no terminal. Uma vez enviado, seu projeto é mapeado e pode ser visualizado, organizado e clonado por qualquer pessoa através do nosso Dashboard.

---

## 🎨 Funcionalidades do Dashboard

* **File Explorer Pro:** Visualize arquivos em uma árvore de diretórios (pastas e subpastas).
* **Syntax Highlighting:** Leitura limpa com cores para diversas linguagens (JS, Solidity, Python, etc).
* **Acesso Público:** Descubra e aprenda com o código de outros desenvolvedores.
* **CLI Sync:** Copie comandos de clonagem com um clique.



  ## 🚀 Funcionalidades Incríveis
* **Interface Hacker-Style**: Visual inspirado em terminais de alta performance.
* **Sincronização Segura**: Vinculada à sua conta Google para autoridade total.
* **Estrutura Inteligente**: Mantém a hierarquia de pastas intacta no upload e download.


---

## 🛠️ Tecnologias Utilizadas

* **Frontend:** Next.js, Tailwind CSS, Lucide Icons.
* **Backend:** Node.js, Express, SQLite.
* **Storage:** File System dinâmico (DNA Storage).
